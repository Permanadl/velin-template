Abbreviations

attr

HTML

If you want to style your text, you should use the following classes instead:
  • .mark will apply the same styles as <mark>
  • .small will apply the same style as <small>
  • .text-decoration-underline will apply the same style as <u>
  • .text-decoration-line-through will apply the same style as <s>
